## Workshop Series

### MCP Development Track (4 weeks)
Master Model Context Protocol development from basics to advanced patterns.

1. **MCP Fundamentals**: Architecture, protocols, first server
2. **Advanced MCP Patterns**: Streaming, batching, error handling
3. **MCP at Scale**: Performance, caching, monitoring
4. **MCP Security**: Auth, encryption, vulnerability prevention

### Glean Power User Track (4 weeks)
Become a Glean expert and maximize your productivity.

1. **Glean Essentials**: Interface, blocks, command search
2. **Advanced Features**: Workflows, agents, AI commands
3. **Customization**: Themes, keybindings, settings
4. **Team Workflows**: Sharing, collaboration, best practices

### Extension Development Track (4 weeks)
Build powerful extensions for the Glean ecosystem.

1. **Extension Basics**: API overview, first extension
2. **UI Components**: Custom views, panels, notifications
3. **State & Data**: Persistence, APIs, external services
4. **Publishing**: Testing, documentation, marketplace submission

## Workshop Format

### Before the Workshop
- **Pre-work**: Short setup guide (15 min)
- **Materials**: Code repository, slides, resources
- **Environment**: Ensure tools are installed
- **Questions**: Submit questions in advance

### During the Workshop
**0:00 - 0:10**: Introduction & Setup
- Welcome and overview
- Quick poll/icebreaker
- Ensure everyone's ready

**0:10 - 0:40**: Part 1 - Core Concepts
- Live coding demonstration
- Explanation of key concepts
- Follow along exercises

**0:40 - 0:50**: Break & Async Exercises
- Complete a short exercise
- Ask questions in chat
- Network with attendees

**0:50 - 1:15**: Part 2 - Advanced Topics
- Building on part 1
- Real-world examples
- Best practices

**1:15 - 1:30**: Q&A & Next Steps
- Open questions
- Additional resources
- Preview next workshop

### After the Workshop
- **Recording**: Shared within 24 hours
- **Code**: Complete examples on GitHub
- **Certificate**: Completion certificate (optional)
- **Follow-up**: Join Discord for continued discussion
